# Phase 5.1 — Self-Regulation Metrik (Δconfidence) Mini-Spec

**AMI-ENGINE:** Soft clamp sonrası confidence değişimi ile "kendini toparlama" ölçümü.

Referans: 09_PHASE_461 (Soft Clamp), B.3 Confidence.

---

# 1. Amaç

- Soft clamp **gerçekten işe yarıyor mu?**
- Hangi profile’da **daha çok iyileştirme** sağlanıyor?
- Motor **kendini regüle edebiliyor mu?**

Metrik: **Δconfidence = confidence_after − confidence_before** (Level 1 soft clamp uygulandığında).

---

# 2. Tanım

- **confidence_before:** İlk seçim (clamp öncesi) confidence.
- **confidence_after:** Soft clamp uygulanmış aksiyon üzerinden hesaplanan confidence.
- **delta_confidence:** confidence_after − confidence_before. Sadece soft_safe_applied=True iken anlamlı.

---

# 3. Çıktı

- **selection_data / out:** `"self_regulation": { "delta_confidence": float | None }`. None = clamp uygulanmadı.
- **Monte Carlo:** mean Δconfidence, std, histogram; isteğe escalation level başına dağılım.

---

# 4. Yorum

- Δconfidence > 0 → clamp sonrası güven arttı (self-regulation işe yarıyor).
- Δconfidence ≈ 0 → davranış değişimi güveni değiştirmedi.
- Δconfidence < 0 → nadir; clamp daha riskli skorlara itmiş olabilir.

Bu spec, Self-Regulation metriğinin referans dokümanıdır.
